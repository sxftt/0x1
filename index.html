Features

With 0x1, you can be the main character. 0x1s best features are:

  • Read and write NFC and RFID tags, as well as emulating them.
  • Receiving and transmitting radio signals.
  • Creating WiFi Networks.
  • Controlling some devices (like TVs).

Examples of usage.

Real life? Too boring.. Watchdogs in real life? Better!

  • With the NFC and RFID reader and writer, you can turn 0x1 into a access card.
  • Best module so far is the radio receiver and transmitter, with it, accessing any garage is simple.
  • Using the WiFi feature, creating WiFi networks is as easy as saying 'Hello world!'.
  • The infrared transceiver allows you to use the 0x1 as a remote, how cool is that?

Components

A special device like this needs special components:

  • Active Buzzer 5V
  • Push buttons (5 buttons)
  • A few resistors
  • RF 868MHz CC1101 Radio Module
  • RFID NFC PN532 Reader
  • Universal PCB 9x15 cm
  • ESP32 WiFi BLE 4.2 DevKitC Compatible Board
  • OLED Graphic Display 128x64px 0.96 inch
  • Lithium Battery Charger TP4056 USB-C
  • Infrared emmiter and receiver
